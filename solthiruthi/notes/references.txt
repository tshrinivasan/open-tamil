# declension
# Accusative form : வை
# Instrumental form: வால்
# Associative form: வோடு
# Dative form: வுக்கு
# genitive form: வின்
# Possessive form: வுடைய
# Locative form: வில்
# Ablative form: விலிருந்து

# Pronouns
e.g. yen
Accusative form: ை
Instrumental form: ஆல்
Associative form: ஓடு
Dative form: உக்கு
Possessive form: உடைய
Locative form: இடம்
Ablative form: இடமிருந்து
